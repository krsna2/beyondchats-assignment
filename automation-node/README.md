# NodeJS Automation (Phase 2)

## Purpose
This module automates article enhancement by:
- Fetching the latest article from Laravel APIs
- Searching Google for related blogs
- Scraping top-ranking articles
- Preparing content for LLM-based enhancement
- Publishing updated articles back to Laravel
